# 🚀 Installation & Usage Guide - TNS Style Page

**Created by TNS**

This guide will help you install, run, and deploy the project on your machine.

---

## 1. Prerequisites (What you need)

Before starting, ensure you have the following installed on your computer:

*   **Node.js**: The engine that runs the code.
    *   Download here: [https://nodejs.org/](https://nodejs.org/) (Choose the "LTS" version).
*   **A Code Editor**: Visual Studio Code is recommended.

---

## 2. Installation

1.  **Unzip the folder** provided by TNS.
2.  Open the folder in your code editor (VS Code).
3.  Open a terminal (in VS Code: `Terminal` -> `New Terminal`).
4.  Run the following command to install dependencies:

```bash
npm install
```

*This may take a few minutes. It downloads necessary libraries like React, Next.js, and GSAP.*

---

## 3. Running the Project (Development Mode)

To see the site and make changes in real-time:

1.  In the terminal, run:

```bash
npm run dev
```

2.  Open your web browser and go to: `http://localhost:3000`

You should see the site! Any changes you make to the code will automatically update the page.

---

## 4. Building for Production (Going Live)

When you are ready to put the site online (e.g., on Vercel or Netlify):

1.  Run the build command to optimize the site:

```bash
npm run build
```

2.  Then, to preview the production version locally:

```bash
npm run start
```

---

## 5. Troubleshooting

*   **"Command not found"**: Ensure Node.js is installed correctly. Restart your computer if necessary.
*   **Port 3000 already in use**: The terminal will usually ask to use another port (like 3001). Just say "Yes" (y).
